P 1.1 Arbeitsumgebung und Grundlagen HTML

Aufgabe 1

Richten Sie sich eine vollständige Arbeitsumgebung ein, in dem Sie z.B. folgende Schritte durchführen:

  • Installieren Sie sich Visual Studio Code oder eine IDE (Entwicklungsumgebung) Ihrer Wahl
  • Installieren Sie sich GIT
  • Legen Sie – falls noch nicht vorhanden – ein GitHub Konto an
  • Legen Sie ein Repository mit dem Namen GIS-WiSe-2020-2021 an
  • Machen Sie sich mit GIT vertraut (wir empfehlen Ihnen die Verwendung einer visuellen Benutzeroberfläche für GIT wie z.B. GitKraken)

Aufgabe 2

Erstellen Sie eine stillose Webstite (reines HTML-Dokument ohne CSS und JS) zu Ihrer Person die mindestens folgende Elemente enthält:

  • die HTML Basisstruktur mit den HTML Tags doctype, html, head, body
  • eine Deklaration der Zeichenkodierung und eines Titels - siehe Überschrift (h tag)
  • ein Bild von Ihnen - siehe Image tag
  • jeweils einen Textabsatz zu Ihrer Person und zu Ihrem Studium - siehe Überschrift (h tag) und Textblock (p tag)
  • einen Link (a tag) auf eine andere Website ihrer Wahl
  • einen Link (a tag) von einem Textblock (p tag) auf einen anderen (Anker → verwenden Sie dafür das id-Attribut in einem anderen HTML Element)

Aufgabe 3

Entdecken Sie die Entwicklungswerkzeuge, die in Ihrem Browser eingebaut sind:

  • Chrome
  • Firefox
  • Safari
  • Opera
  • Edge

Die Entwicklungswerkzeuge sind extrem hilfreich im Laufe des gesamten Semesters (und darüber hinaus). Wir empfehlen Ihnen sich hier einzulesen und diese Funktionalitäten in Ihrem Browser auszuprobieren.
